# Spaghetti Chat

Simple chat app built with [Next.js](https://nextjs.org/) and [tailwindcss](https://tailwindcss.com/)

## Running the dev server

```bash
npm run dev
```
Open [http://localhost:3000](http://localhost:3000) with your browser.


## Running on production

Compile the files
```bash
npm run build
```
Start the server
```bash
npm run start
```
